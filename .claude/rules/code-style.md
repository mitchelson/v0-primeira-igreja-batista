# Estilo de Código

- TypeScript strict, sem `any`
- ES modules (import/export)
- Componentes React: function components com arrow functions
- Hooks customizados em `hooks/`
- Alias `@/` para imports da raiz
- Strings em português para UI (labels, mensagens, placeholders)
- Nomes de variáveis/funções em inglês, textos de UI em português
